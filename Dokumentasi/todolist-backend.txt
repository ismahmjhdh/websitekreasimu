=== SETUP PROJECT BACKEND (LARAVEL) ===

Requirement:
- PHP 8.2.x (WAJIB, jangan 8.5)
- Composer
- Git
- MySQL (kalau pake DB)
- Apache / Nginx / Laragon / XAMPP / WSL (bebas)

--------------------------------------------------

1. Clone repository
git clone <URL_REPO>
cd BACKEND

--------------------------------------------------

2. Pastikan versi PHP
php -v

Harus:
PHP 8.2.x

Kalau masih 8.5, downgrade dulu.

--------------------------------------------------

3. Install dependency
composer install

NOTE:
- Jangan push folder /vendor ke Git
- Folder ini auto ke-generate di sini

--------------------------------------------------

4. Setup environment
copy .env.example .env

--------------------------------------------------

5. Generate APP KEY
php artisan key:generate

--------------------------------------------------

6. (Optional) Setup Database
Edit file .env:
DB_DATABASE=nama_db
DB_USERNAME=root
DB_PASSWORD=

Lalu jalankan:
php artisan migrate

--------------------------------------------------

7. Jalankan server
php artisan serve

Akses:
http://127.0.0.1:8000

--------------------------------------------------

Common Error Fix:
- php artisan optimize:clear
- pastikan extension PHP aktif (mbstring, openssl, pdo, xml)

--------------------------------------------------

If error:
- Cek versi PHP
- Cek composer.json
